# TwitchChatBot
Twitch chat bot that interacts with an AI on chat, and sends ads at every 5 minutes
## Pre Requisites

1. node.js
2. tmi.js
3. ems
4. openai

## Setting up

### npm: 
`npm install`

### Add your tokens:

### Add your twitch bot user, bot oauth, and desired channel in `config.js`
### Add your OpenAI API and ORG keys in `config.js`
